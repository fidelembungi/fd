À qui s'adresse cette formation ?

Cette formation est destinée aux débutants, passionnés ou professionnels qui souhaitent apprendre à créer des sites web modernes et dynamiques, pas à pas.

Contenu de la formation

HTML & CSS

  • Structure de pages avec HTML5
  • Mise en forme avec CSS3
  • Mise en forme avec Bootstrap
  • Animation avec Animate__CSS
  • Responsive Design pour mobile

JavaScript

  • Syntaxe et logique de base
  • DOM : manipulation d’éléments HTML
  • Événements, fonctions, conditions, boucles
  • Projets pratiques (ex : todo list, etc.)
  • Récuperer l'heure automatique via Ajax (Intro)

JavaScript avec AJAX

  • Comprendre le fonctionnement d’AJAX
  • Envoyer/recevoir des données sans recharger la page et Utiliser `fetch()`
  • Créer des interfaces dynamiques.
  • Exemples pratiques

PHP & MySQL

  • Introduction à PHP
  • Connexion à une base de données MySQL
  • Insertion, affichage, modification et suppression de données
  • Systèmes de connexion simples

Hébergement & mise en ligne

  • Structuré le site
  • Préparer votre site pour la mise en ligne
  • Utiliser un hébergeur gratuit ou payant
  • FTP, nom de domaine, base de données en ligne
  • Exemple pratique

Modalités

Formations en présentiel ou en ligne, Outils gratuits et Documents des formations sous plusieurs types, selon votre rythme. Possibilité de coaching personnalisé, assistance après formation et Accompagnement à vos premiers réalisations.
